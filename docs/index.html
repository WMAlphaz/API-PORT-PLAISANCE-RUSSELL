Documentation de l'API Port de Plaisance Russell

Port de Plaisance Russell

Cette API permet au membre de la capitainerie de gérer les réservations des clients pour une passerelle donnée, ainsi que de gérer les utilisateurs de l'API (création, modification et suppression). Elle permet également de récupérer les informations des catways ou des réservations.

Fonctionnement

Connectez-vous d'abord avec vos identifiants. Vous arriverez sur le dashboard. Depuis celui-ci, vous pourrez gérer les utilisateurs, les réservations ainsi que créer des catways.

Pour gérer les catways, cliquez sur "Cateways" présent dans la navigation. Vous aurez alors la liste des catways. Vous pourrez ajouter des cateways, voir les détails et voir les réservations. . Vous pourrez aussi modifier la description du catway via les détails.
